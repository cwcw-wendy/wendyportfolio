About project

The New Taipei Love Book App mainly provides New Taipei City Library readers with real-time collection inquiry, library service push notifications (reservation pick-up, borrowing expiration reminder, and overdue borrowing push notifications), reservation renewal, and mobile borrowing card services.

Project Goal

As a frequent visitor to the library, I was delighted to be able to borrow books through a mobile app after borrowing them from a regular computer. The goal of this project is to redesign the homepage and my bookshelf pages to guide lost users in the right direction. Therefore, in order to improve the app usage rate and provide a better user experience, the team will redesign the original five main functional pages.
